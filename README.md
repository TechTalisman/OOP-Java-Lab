# Object-Oriented Programming (Java) Lab

This repository contains the Java programs completed during the **Object-Oriented Programming (OOP) Laboratory** course in the **4th Semester of B.Tech Computer Science & Engineering**.

Each folder represents a separate lab session and contains the programs implemented during that practical class.

## Topics Covered

- Java Basics
- Classes & Objects
- Constructors
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Exception Handling
- File Handling
- Multithreading

## Repository Structure

```
Lab-01/
Lab-02/
...
Lab-10/
```

## Run a Program

```bash
javac FileName.java
java FileName
```
